# LFMDateDifferenceToString

## Description

This is a simple library that receives a date difference and returns it as a string.

For example, if you send a Date Difference of **300s**, the library will return **5m**.

Other examples:

* 7,200s - 2h
* 88,000 - Yesterday
* 172,800 - 2 days ago
